# 2D-net-design-
Working on packaging for a chocolate bar project. Created my design in onshape so thought i may as well upload it.  A basic model which i printed out and assembled. Model designed to hold ten kitkat sized chocolate bars while hiking or cycling so the packacking allows it to be stored inside a bike bottle holder or a backpack bottle holder.
